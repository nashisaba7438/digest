📋 APIキーの取得方法（無料・3分）
1
aistudio.google.com/apikey を新しいタブで開く
2
Googleアカウントでログイン（Gmailのアカウントでそのまま使えます）
3
青い 「Create API key」 ボタンをクリック → 「Create API key in new project」 を選択
4
「AIza...」 から始まる長い文字列が表示される → コピーして下の入力欄に貼り付け
⚠️ 無料枠の制限について
現在Googleの無料枠は 1日20回 までに制限されています。たくさん使う場合は「⚡ 制限解除」タブを参照してください（追加費用なし）。

⚡ 1日20回 → 1500回以上に増やす方法
カードを登録するだけで制限が一気に緩和されます。
使わなければ課金は一切ゼロです。月数円〜数十円程度の使用量なので実質無料です。
1
aistudio.google.com/apikey を新しいタブで開く
※ 同じGoogleアカウントでログインしてください
2
APIキーの一覧の右側に 「Set up billing」 というリンクがある → クリック
※ すでに設定済みの場合はこのリンクがありません（設定完了済みです）
3
Google Cloud の請求設定画面に移動する → アカウント名・国・クレジットカードを入力して 「Submit and enable billing」 をクリック
※ $1 の仮請求が入りますがすぐ返金されます。本請求ではありません
4
AI Studioに戻り、再度 「Set up billing」 をクリック → 「Link a billing account」 → 作成した請求先アカウントを選択 → 「Set account」
※ このリンク手順を忘れると制限が解除されません
5
APIキー一覧の表示が 「Paid account」 になれば完了！
制限が 1日1500回以上 に緩和されます（すぐ〜数分で反映）
💡 本当に無料なの？
DIGESTの使い方（1日3回×6カテゴリ程度）なら月 $0.01〜$0.10（約1〜15円）未満です。Google Cloudには $300 の無料クレジットもあるので、しばらくは実質完全無料です。
💡 いつでも解約できる？
はい。Google Cloudのコンソールからいつでも請求先アカウントを削除できます。
